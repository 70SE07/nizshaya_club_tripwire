# Маркетинговый разбор лендинга — ветка main

## Экран 1 — Оффер (Hero)
Главное предложение. Цепляет заголовком, сразу показывает цену ($29), даёт две кнопки (купить / посмотреть программу). Снизу триггеры доверия: гарантия 14 дней, анонимно, доступ навсегда.

## Экран 2 — Усиление боли (Pain)
Давит на 4 конкретные боли целевой аудитории. Каждая — карточка с иконкой. В конце — цитата-отзыв от ученицы, которая усиливает эмоцию ("я чувствовала себя обслугой").

## Экран 3 — Трансформация (Solution)
"Было → Стало". 5 пар: проблема зачёркнута → результат. Визуально показывает путь. Внизу — позитивный отзыв ("он сказал: где ты этому научилась?!").

## Экран 4 — Программа (Program)
Конкретика: 5 дней, каждый — конкретный навык. Снимает вопрос "а что внутри?". Каждый день — своя цветовая тема.

## Экран 5 — Эксперт (Expert)
Социальное доказательство через авторитет. Катя — сексолог-практик. 3 карточки: диплом, 700+ учениц, экологичный подход. Позиционирование: "не блогер, а профессионал".

## Экран 6 — Оффер повторный (Offer)
Повтор предложения с полным списком что входит (6 пунктов + 2 бонуса). Старая цена зачёркнута ($79 → $29). Триггеры: гарантия, анонимность, свой темп.

## Экран 7 — FAQ
Снятие возражений. 6 вопросов, каждый — конкретный страх: увидят списание, стыдно, рвотный рефлекс, одинока. Аккордеон — раскрываются по клику.

## Экран 8 — Финальный CTA + Футер
Дожим. Эмоциональный заголовок + последняя кнопка "Начать за $29". Создаёт ощущение упущенной возможности.

---

## Структура по маркетинговой формуле
1. **Оффер** → 2. **Боль** → 3. **Трансформация** → 4. **Программа** → 5. **Эксперт** → 6. **Оффер (повтор)** → 7. **Возражения** → 8. **Дожим**

Классическая воронка: зацепил → надавил → показал результат → доказал → продал → закрыл возражения → дожал.
