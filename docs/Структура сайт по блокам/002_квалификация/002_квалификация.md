# 2. Квалификация

## Говорит
Кому

## ЦКП
Посетитель узнаёт себя: "это про меня". Фильтрует аудиторию — целевая остаётся, нецелевая уходит.

## Вопрос посетителя
"Это для меня?"

## Заметки
- По Воскресенскому: надзаголовок (Pre-headline) выполняет функцию жёсткой сегментации. Цель — отсечь нецелевых и сфокусировать целевых.
- Формулировки типа "Для тех, кто..." работают как фильтр, повышающий релевантность.
- Квалификация и Эскалация — разные вещи. Квалификация говорит КОМУ. Эскалация говорит ЗАЧЕМ.
- Можно совмещать с оффером в один экран, а можно разделять.
